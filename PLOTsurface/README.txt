Function: plot surface

Run format:
	PlotSurface

NOTE: 
update ellipsoid RBF information file and corresponding PQR file to this folder before run this script.

where ellipsoid RBF information file contains:
*_Weight.txt
*_DecayRate.txt
*_Center.txt
*_Angle.txt

For example:
ADP_Weight.txt
ADP_DecayRate.txt
ADP_Center.txt
ADP_Angle.txt

PQR file:
*.pqr
Eg.
ADP.pqr
